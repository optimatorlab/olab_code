# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any package in this repository,
please report it privately rather than opening a public issue — use GitHub's
[private vulnerability reporting](https://github.com/optimatorlab/olab_code/security/advisories/new)
for this repo.

Please include:
- The affected package(s) and version/commit.
- Steps to reproduce, or a minimal example.
- The potential impact as you understand it.

We'll acknowledge reports as soon as we're able and follow up with a fix or
mitigation plan.
